This project contains Android-JBridge server side scripts to deploy RESTFul web services for a shop application under JBoss.

You can use this product under GPL license.

To deploy server EAR file, edit local.properties and set the jboss_home variable. 

Then build the project using Apache Ant.

Shop storefront URL: http://127.0.0.1:8080/JBridge/Shop

Web services URL: http://127.0.0.1:8080/JBridge/RestEasy/products
